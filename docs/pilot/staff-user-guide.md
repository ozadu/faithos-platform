# Staff User Guide

Staff should use the browser UI rather than Swagger.

- Log in at `/login`.
- Create documents at `/documents/create`.
- View routed work in `/inbox`, `/my-work`, `/my-tasks`, and
  `/pending-approvals`.
- Use document detail pages for submit, forward, return, receive, timeline, and
  attachment testing.
- Use `/notifications` for database notification records.
- Submit trial feedback at `/feedback`.
- Ask an administrator to triage urgent issues at `/admin/pilot-issues`.
