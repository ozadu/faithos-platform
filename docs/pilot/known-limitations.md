# Known Limitations

- FaithOS v0.9.0 is for controlled pilot testing, not broad SaaS launch.
- Feedback is lightweight pilot triage, not a full support desk.
- Backup scripts are operator-run helpers, not an automated backup platform.
- UAT runner results are stored in local browser state for this sprint.
- Permission audit highlights obvious risks but is not formal governance.
- Email uses development SMTP/Mailpit unless a production provider is configured.
- SMS, WhatsApp, payments, AI, HR, finance, inventory, and asset modules are not
  included.
