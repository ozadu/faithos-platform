# Go-Live Readiness Checklist

- Pilot readiness complete.
- Production readiness reviewed.
- Setup pack complete or explicitly accepted with risks.
- Backup restore drill completed.
- Demo credentials rotated or removed.
- Real SMTP provider configured outside Mailpit.
- Operational owner assigned for support.
- Known limitations documented.
- Open critical pilot issues closed or accepted.
- Release approval recorded by the human release owner.
