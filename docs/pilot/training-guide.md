# Pilot Training Guide

## For staff

1. Log in at `/login`.
2. Open `/documents/create`.
3. Enter title, subject, body, category, priority, and confidentiality.
4. Save a draft if the document is not ready.
5. Upload supported attachments from the document detail page.
6. Submit the document when ready.
7. Check Inbox, My Work, My Tasks, and Notifications daily.
8. Approve, return, or reject workflow tasks only after reviewing the document.
9. Use Reports if your role allows it.
10. Submit feedback at `/feedback`.

## For administrators

1. Create or verify the first admin account.
2. Complete organization settings.
3. Create departments and assign heads.
4. Create/import users.
5. Assign roles and departments.
6. Review role permissions.
7. Configure document types and workflow assignments.
8. Review pilot feedback and issues.
9. Run backup/restore checks with the technical installer.
10. Review known limitations before go-live.
