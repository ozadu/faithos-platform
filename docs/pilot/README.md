# FaithOS Pilot Trial Pack

This folder contains the Sprint 8 pilot deployment and real-world trial pack.
It is intended for internal administrators, pilot coordinators, and technical
operators preparing FaithOS for a controlled trial.

Primary browser entry points:

- `/uat`
- `/admin/pilot-deployment`
- `/admin/pilot-setup-pack`
- `/admin/demo-credentials`
- `/feedback`
- `/admin/feedback`
- `/admin/pilot-issues`
- `/admin/onboarding-checklist`

Do not treat this pack as a production certification. Production email,
automated backup scheduling, external monitoring, and advanced BI remain future
work.
