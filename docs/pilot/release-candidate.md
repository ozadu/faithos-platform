# FaithOS v0.9.0 Pilot Release Candidate

FaithOS v0.9.0 is a controlled pilot release candidate. It is intended for one
church or organization to test real document routing, workflows, reports,
notifications, feedback, and operational recovery before wider rollout.

Before pilot kickoff:

- Complete `/pilot/checklist`.
- Run `/pilot/uat`.
- Verify `/admin/deployment-readiness`.
- Verify backup and restore using the documented scripts.
- Train administrators, office secretaries, and department heads.

Do not treat this release as broad public SaaS production. It is stable enough
for supervised pilot testing with clear support and rollback procedures.
