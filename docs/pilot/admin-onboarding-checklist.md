# Admin Onboarding Checklist

1. Log in with the seeded demo administrator.
2. Review organization profile, timezone, contact details, and branding.
3. Confirm departments, users, roles, and permission assignments.
4. Confirm document types and workflow assignments.
5. Review system settings, audit log, and pilot readiness.
6. Submit one document, execute one workflow, and review notifications.
7. Open reports and export at least one CSV file.
8. Run a backup drill and record evidence in the pilot issue tracker.
9. Explain the feedback process to pilot staff.
10. Mark completed items at `/admin/onboarding-checklist`.
