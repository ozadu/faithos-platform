# Organization Setup

Complete `/admin/organization` before adding pilot users.

## Required fields

- Name
- Slug
- Email
- Phone
- Address
- Country
- Timezone

## Steps

1. Log in as an administrator.
2. Open `/admin/organization`.
3. Enter the official pilot organization name.
4. Enter a stable slug used for pilot identification.
5. Add the office email and phone number.
6. Add the physical address and country.
7. Confirm the timezone is correct.
8. Save changes.
9. Reopen `/pilot/checklist` and confirm organization readiness is complete.

## Expected result

Organization profile readiness has no missing required fields.
