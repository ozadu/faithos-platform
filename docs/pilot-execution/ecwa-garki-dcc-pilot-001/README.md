# FaithOS Pilot Test 001 Execution Package

Pilot organization: ECWA Garki DCC  
Pilot package: FaithOS Pilot Test 001

## Recommended usage

- Print the Markdown checklists for meetings and sign-off.
- Copy the CSV sheets into Excel or Google Sheets for daily tracking.
- Keep one named pilot owner responsible for collecting all completed records.
- Do not record passwords, JWT secrets, database URLs, SMTP passwords, or private tokens in these documents.
- Any unresolved critical issue should block go-live or wider rollout.
