## Severity definitions

- Critical: blocks login, document routing, approval, data safety, or recovery.
- High: serious operational issue with a workaround.
- Medium: slows users but does not block pilot work.
- Low: cosmetic, wording, minor usability, or documentation issue.

## Status definitions

- New: reported but not reviewed.
- In Review: being investigated.
- Fixed: corrected and ready for retest.
- Closed: verified and accepted.
- Accepted: known limitation accepted for the pilot.
