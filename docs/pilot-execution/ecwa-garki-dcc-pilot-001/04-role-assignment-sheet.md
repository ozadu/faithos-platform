## Permission review rules

- Only trusted administrators should manage users, roles, permissions, and system settings.
- Report access should be limited to users who need management visibility.
- Sensitive permissions must be reviewed in `/admin/permission-audit`.
- Remove unexpected sensitive access before live pilot usage.
